# 🌙 AI 수면 코치 (Sleep-AI Coach)

이 프로젝트는 Streamlit을 사용하여 구현된 AI 기반 수면 패턴 최적화 코칭 서비스입니다. 사용자가 입력한 다양한 생체 및 환경 데이터를 기반으로 깊은 수면 비율을 예측하고, 맞춤형 환경 조절 및 개입 방안을 추천합니다.

## 핵심 분석 및 서비스 목표
1. **다중 데이터 통합:** 카페인, 운동, 온도, 소음, 심박수 등 5가지 핵심 데이터를 통합 분석합니다.
2. **수면 패턴 분석:** 입력값을 기반으로 "깊은 수면(Deep Sleep) 비율"을 예측하여 수면 품질을 정량화합니다.
3. **맞춤형 개입:** 잠자리에 들기 1시간 전에 온도, 조명, 음악 재생 등 최적화 개입을 시뮬레이션하여 추천합니다.

## 실행 방법 (Streamlit Cloud 배포 기준)
1. **GitHub 저장소 Fork/Clone:** 이 저장소를 복사합니다.
2. **Streamlit Cloud 연결:** Streamlit Cloud (share.streamlit.io)에 접속하여 이 저장소를 연결하고, 메인 파일(`sleep_coach_app.py`)을 지정합니다.
3. Streamlit Cloud가 `requirements.txt`를 읽어 자동으로 앱을 배포합니다.
